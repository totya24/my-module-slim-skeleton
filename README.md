# Slim3 skeleton with modules

Simple Boilerplate based on [Slim3 Framework](https://www.slimframework.com/).

## Features
- [TWIG templates](https://twig.symfony.com/)
- [Whoops Error Reporting](https://github.com/dopesong/Slim-Whoops)
- [Slim Tegar DI](https://packagist.org/packages/ibnusyuhada/slim-tegar)
- [Medoo database library](https://medoo.in/)
- [UtilPHP](http://brandonwamboldt.github.io/utilphp/)
